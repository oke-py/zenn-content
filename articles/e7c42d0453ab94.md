[OutSystems Advent Calendar 2025](https://adventar.org/calendars/12138)、3日目の記事です。

## はじめに

こんにちは、おけぽよです。2023年からコミュニティでの登壇を中心に活動しています。「自己紹介で乃木坂46の話をする人」「乃木坂46のTシャツを着ている人」に心当たりがあれば、それは私だと思います。

今年は **OutSystems Expert Developer (O11, ODC)** の認定を取得できました。Expert認定に至るまでには、広範な知識と複数の試験への合格が求められます。本記事では、効率よく知識を定着させる「ChatGPTとNotebookLMを活用した学習法」を紹介します。

## OutSystems Expert Developerとは

OutSystems Expert Developerは単一の試験ではなく、以下の認定をすべて取得することで自動的に付与される称号です。

* Associate Developer
* Architecture Specialist
* Web Developer Specialist
* Front-end Developer Specialist
* Mobile Developer Specialist
* Security Specialist

## なぜExpert認定を目指したのか

### きっかけはコミュニティからの後押し
きっかけは、コミュニティ運営や現役MVPの方からいただいた「MVPを目指しませんか？」という一言でした。これまでコミュニティでの登壇は続けてきましたが、MVPを目指すにあたり、まずは客観的なスキル証明としてExpert認定の取得を決意しました。

### 資格収集は「趣味」
私はもともと、AWSやCNCF（Cloud Native Computing Foundation）などの資格を集めるのが趣味です。「実務さえできれば資格は不要」という意見も一理ありますが、私にとっては **新しい技術分野の体系的な知識を効率よくインプットする手段** として、資格試験を楽しんでいます。

資格収集には慣れていますが、今回は対象範囲が広大です。そこで、この趣味の活動をさらにアップデートするために、**AI（ChatGPTとNotebookLM）をフル活用した新しい学習スタイル**を試してみることにしました。
